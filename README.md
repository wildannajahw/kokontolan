# Frame Fitter

Website based tools to fit twibbons / photo frame.
Website is made for testing canvas and thus is made entirely of html-css-jss. Web features HTML5 Canvas, and uses JQuery, JQueryUI, and Bootstrap for faster development.

## Features
1. Rotate, Scale images
2. Save images
3. Support for touch based gesture
